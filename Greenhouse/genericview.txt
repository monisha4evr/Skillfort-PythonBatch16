Another Method:
View:
from django.views.generic import TemplateView

class ClassHome(TemplateView):
    template_name = "class/class_home.html"

    def get_context_data(self, **kwargs):
        context = super().get_context_data(**kwargs)

        context["name"] = "mm"
        context["age"] = 99

        return context


class ClassHome(TemplateView): → Create a class that inherits from TemplateView.
template_name → Specifies which HTML file to render.
get_context_data() → Used to send data from the view to the template.
super().get_context_data(**kwargs) → Gets the default context from the parent TemplateView.
context["name"] = "mm" → Adds a custom value to the context.
return context → Sends all context data to the template.

urls:
from django.urls import path
from .views import ClassHome

urlpatterns = [
    path("class-home/", ClassHome.as_view(), name="class_home"),
]
