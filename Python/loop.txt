Loops
Python uses loops to execute a block of code repeatedly.
while loop
for loop

for loop:

Used to iterate over a sequence (like a list, tuple, string, or range). 
It executes the code once for each item in that sequence.

Common Use: Iterating a fixed number of times using the range() function.
fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

2. The while LoopRepeatedly executes a block of code as long as a specified condition remains True
Common Use: When the number of iterations isn't known beforehand.

i = 1
while i < 6:
    print(i)
    i += 1 

3. Loop Control Statements:
    These statements change the execution from its normal sequence:
    break: Stops the loop entirely, even if the condition is still true.
    continue: Skips the current iteration and moves to the next one.
    pass: Acts as a placeholder when a statement is syntactically required but no action is 
    needed.

Convert the rows of a 2D grid (matrix) into columns, and columns into rows.

matrix = [[1, 2], [3, 4]]
# Create an empty matrix to hold the result
transpose = [[0, 0], [0, 0]] 
for i in range(len(matrix)):
    for j in range(len(matrix[0])):
        transpose[j][i] = matrix[i][j]
print(transpose)  # Output: [[1, 3], [2, 4]]
