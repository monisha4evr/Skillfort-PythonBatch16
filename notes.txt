Disadvantage:   
    1. Slow Execution Speed:
        1. Dynamic Typing
        2. Everything is an Object
        3. Garbage Collection

    Partially interpreter
    Python is called partially interpreted because Python source code is first compiled into bytecode, and then the Python Virtual Machine interprets and executes that bytecode line by line.

    Python is slower because it is an interpreted and dynamically typed language.
It performs runtime type checking, object handling, and garbage collection, which add extra processing overhead.

2. High Memory Consumption:
    Python consumes more memory because everything in Python is treated as an object.
Each object stores additional information like datatype, reference count, and metadata.
Python also uses dynamic typing and garbage collection, which require extra memory management.

Rules in Python Syntax
    1. Python is case-sensitive
    2. Indentation is important
    3. No need for {} braces
    4. Statements end automatically in a new line

Input and output Operation:
    In Python, input and output (I/O) operations allow a program to interact with the user by receiving data and displaying results.

1. Output Operation (print())The print() function sends data to the screen. It can print strings, variables, or the results of expressions.
        1.print("Hello, World!")

        2.  name = "Alice"
            age = 25
            print("Name:", name, "Age:", age)

        3. print(f"Next year, {name} will be {age + 1} years old.")

2. Input Operation (input())The input() function pauses program execution and waits for the user to type something into the terminal.

    Crucial Rule: The input() function always returns the data as a String (str), even if the user types a number.

Variables:
    Variables are used to store data.its like a container

    Rules:
    1. Allowed Characters: Names can only contain letters (A-Z, a-z), digits (0-9), and underscores (_).
    2. Starting Character: Names must start with a letter or an underscore. They cannot start with a number.
    3. No Spaces: You cannot use spaces inside a variable name.
    4. No Special Characters: Symbols like @, $, %, -, or ! are strictly forbidden.
    5. Reserved Keywords: You cannot use Python's built-in keywords (such as if, for, while, class, or import) as variable names.
